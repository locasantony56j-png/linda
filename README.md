# Bristy Health Care 🏥

A futuristic medical and educational platform for students in Bangladesh, featuring medicine search, medical dictionary, and study tools.

## Features ✨

- **Medicine Database**: Search for medicines by name, generic, or company.
- **Medical Dictionary**: Comprehensive dictionary for medical terms and academic terms.
- **AI Assistant**: Powered by Gemini AI to answer medical queries and analyze scans.
- **Scan & Diagnosis**: Analyze medical reports or symptom photos using AI.
- **Study Tools**: Flashcards, quizzes, and study notes for medical students.
- **Emergency Guide**: Quick access to emergency medical procedures.
- **PWA Ready**: Installable on mobile and desktop for offline-like experience.

## Tech Stack 🛠️

- **Frontend**: React 19, TypeScript, Tailwind CSS 4.
- **Animations**: Motion (formerly Framer Motion).
- **Icons**: Lucide React.
- **AI**: Google Gemini API.
- **Storage**: LocalStorage (Mock Firebase for local-only mode).

## Getting Started 🚀

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/bristy-health-care.git
   cd bristy-health-care
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file in the root directory and add your Gemini API key:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Build for production:
   ```bash
   npm run build
   ```
